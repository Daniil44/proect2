# Запуск и сборка

## Требования

- Node.js
- npm или pnpm
- PostgreSQL
- переменные окружения из `.env.example`

## Переменные окружения

Пример находится в `.env.example`.

Обязательные значения:

- `HOST` - host для Fastify.
- `PORT` - порт HTTP сервера.
- `DATABASE_URL` - строка подключения PostgreSQL.
- `JWT_SECRET` - секрет для подписи JWT.

## Установка зависимостей

```bash
npm install
```

или

```bash
pnpm install
```

## Сборка клиента

```bash
npm run build
```

Vite собирает клиент из `views` в `public/dist`.

Важно: Fastify раздает именно `public/dist`, поэтому после изменений в `src/client` нужно пересобирать проект.

## Запуск сервера

```bash
npm start
```

Сервер:

- инициализирует подключение к PostgreSQL;
- создает таблицы игроков и сообщений;
- раздает статический frontend из `public/dist`;
- поднимает Socket.IO.

## Частые проблемы

### Сервер не стартует

Проверьте `DATABASE_URL` и доступность PostgreSQL.

### Клиент показывает старое поведение

Скорее всего, не выполнена сборка после изменения `src/client`.

### Login возвращает 400

Ник должен быть от 3 до 25 символов и состоять только из латиницы, цифр и `_`.
