# Документация проекта

Проект - браузерная multiplayer-демка на Three.js, Fastify, Socket.IO и PostgreSQL.
Клиент отображает игроков кубами на плоскости, показывает ники и всплывающие сообщения над игроками, а сервер хранит пользователей и историю чата.

## Основной пользовательский сценарий

1. Пользователь открывает страницу.
2. Вводит ник в login-панель.
3. Клиент отправляет `POST /auth/login`.
4. После успешного ответа клиент подключается к Socket.IO с JWT token.
5. Сервер отправляет `init` с текущими игроками и историей чата.
6. Игрок управляет кубом клавишами `WASD` или стрелками.
7. Сообщения отправляются в чат и появляются bubble-лейблом над игроком.

## Текущий frontend entrypoint

Главная точка входа клиента: `src/client/client.js`.

Файл intentionally маленький: он связывает UI, сцену, socket-контроллер, input-контроллер, players manager и чат.
